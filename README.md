# YouTube Live Chat Analyzer

Анализ чата YouTube в реальном времени. Сбор сообщений, метрики эфира, топ участников, экспорт CSV.

## Возможности

- 📊 **Сбор чата** — все сообщения из прямого эфира в реальном времени
- 👥 **Топ участников** — самые активные зрители чата
- 📈 **Метрики эфира** — текущее количество зрителей, максимум зрителей, количество лайков
- 📥 **Экспорт CSV** — сохраните чат для дальнейшего анализа
- 🖥️ **Отдельное окно Dashboard** — вынесите статистику на второй монитор
- 🔄 **Автообновление** — данные обновляются каждые 3 секунды

## Установка

### Firefox (временно, до публикации в магазине)

1. Скачайте или склонируйте репозиторий
2. Откройте Firefox и перейдите на страницу `about:debugging`
3. Нажмите **"Загрузить временное дополнение"**
4. Выберите любой файл из папки расширения (например, `manifest.json`)
